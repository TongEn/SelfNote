总体计算原则

计量单位-数量

总量_零散
当日需完成总量

通知量总和

当日已完成总量

当日完成数量之和

总量_箱
当日需完成总量

通知量向上取整箱数总和

当日已完成总量

当日完成数量向上取整箱数之和

总量_板
当日需完成总量

通知量向上取整板数总和

当日已完成总量

当日完成数量向上取整板数之和

拆量_零散
当日需完成总量

通知量-通知量向下取整箱数 剩余量总和

当日已完成总量

当日完成数量-当日完成数量向下取整箱数 剩余量总和

拆量_箱
当日需完成总量

通知量-通知量向下取整板数的剩余量再向下取整的箱数综合

当日已完成总量

当日完成数量-当日完成数量向下取整板数的剩余量再向下取整的箱数综合

拆量_板
当日需完成总量

通知量向下取整板数总和

当日已完成总量

当日完成数量向下取整板数之和

计量单位-储位数

Note

储位类型为平库10、重型11、立库12按板储位计算

非板储位按上架量计算,有箱算箱储位,没有箱算拆零

一板一個儲位,换算为几板就是几个储位

剩余的再換算箱，大于等于1箱时，視為一個儲位

剩余的零散，大于等于1时，視為一個儲位

拆量_零散
当日需完成总量

有零散量的通知明细行数总和

当日已完成总量

有零散量的完成明细行数总和

拆量_箱
当日需完成总量

除去板数后有箱量的通知明细行数总和

当日已完成总量

除去板数后有箱量的完成明细行数总和

拆量_板
当日需完成总量

通知量向下取整板数总和

当日已完成总量

当日完成数量向下取整板数之和

计量单位-单据明细

总量_零散
当日需完成总量

通知明细行数之和

当日已完成总量

当日完成明细行数之和

总量_箱
当日需完成总量

通知明细行数之和

当日已完成总量

当日完成明细行数之和

总量_板
当日需完成总量

通知明细行数之和

当日已完成总量

当日完成明细行数之和
